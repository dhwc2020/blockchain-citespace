# blockchain-visualized-analysis
主要功能是完成区块链公链数据采集与可视化分析

## 模块说明
```
blockchain-visualized-analysis
├── blockchain-visualized-analysis-gateway         // 网关模块 [8080]
├── blockchain-visualized-analysis-auth            // 认证中心 [9200]
├── blockchain-visualized-analysis-analyse         // 数据分析服务
├── blockchain-visualized-analysis-collection      // 数据采集服务
├── blockchain-visualized-analysis-links           // 链接管理服务
├── blockchain-visualized-analysis-member          // 会员服务
├── blockchain-visualized-analysis-common          // 公共模块
├──pom.xml                // 公共依赖
```
